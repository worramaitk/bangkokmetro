แผนที่ระบบขนส่งมวลชนในกรุงเทพและปริมณฑล

Thai ภาษาไทย

นี่คือแผนที่ของระบบรถไฟฟ้าและระบบขนส่งมวลชนอื่นๆ ในกรุงเทพฯ และปริมณฑล แผนที่นี้สร้างโดยใช้โปรแกรม Inkscape
แผนที่นี้ใช้ฟอนต์ Monda และ Sarabun

English ภาษาอังกฤษ

This is a Map of mass rapid transit systems in Bangkok Metropolitan Region. I created this map with Inkscape
This map uses the typefaces Monda and Sarabun.
